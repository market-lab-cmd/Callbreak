# Project Specification

## Positioning
Educational documentation repository.

## Allowed
- Gameplay explanations
- Strategy notes
- Beginner guides

## Forbidden
- Gambling / betting content
- "earn money" language
- Promotional or referral content
- SEO manipulation

## AI Rules
Content must be:
- Original
- Neutral
- Informational
- Non-promotional

## Article Generation Rules
- Each article must have a sequential number (e.g., 001, 002, 003...)
- Every generated article must be added to README.md with a link

## Mandatory Checklist
- No money-related language
- No promotional intent
- No duplicate content
- Must be educational
